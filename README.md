# The Art Zone

URL: https://tranquil-muffin-f9a7d9.netlify.app/

The Art Zone is a website where you can see wonderful drawings from Andrew Abdulaziz, Einstein Nnah, and Keshika Somaskanthar. Users are greeted at the home page and have the options to navigate the about section, gallery, and artists section.

![Home](./readmeimg/Screenshot%202025-12-22%20194544.png)

## About

Here users can learn more about The Art Zone and what our mission is. You can navigate to this section by pressing the "about" button.

![About](./readmeimg/Screenshot%202025-12-22%20194557.png)

## Gallery

Here you can see the drawings and who drew them. You can get to this section by pressing the "gallery" button. You can navigate through each drawing by either scrolling or pressing on the drawing in the sidebar.

![Gallery](./readmeimg/Screenshot%202025-12-22%20194611.png)

## Artists

Here you can learn more about each of the main 3 artists. You can navigate to this section by pressing the "artists" button.

![Artists](./readmeimg/Screenshot%202025-12-22%20195128.png)
